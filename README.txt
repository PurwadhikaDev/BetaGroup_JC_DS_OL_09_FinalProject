Pada folder ini hasil dari model dan analisis dibagi menjadi beberapa file sebagai berikut:
a. 1_Problem Statement & Data Understanding: Berisi mengenai penjelasan masalah dan pemahaman akan data
b. 2_Exploratory Data Analysis: Berisi mengenai analisis karateristik dari dataset yang digunakan
c. 3_Preprocessing: Berisi mengenai pengolahan awal data
d. 4_Data Analytics: Berisi mengenai analisa sehingga didapatkan pemahaman dan rekomendasi dari data
e. 5_Model ML_Residential: Berisi mengenai model Machine Learning pada bagian "Residential"
f. 5_Model ML_Condominium: Berisi mengenai model Machine Learning pada bagaian "Condominium"
g. 6_Conclusion & Recomendation: Berisi mengenai kesimpulan dan saran
